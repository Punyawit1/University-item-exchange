ระบบแลกเปลี่ยนสิ่งของในมหาวิทยาลัย 📦
=================================================================



โปรเจคนี้เป็นระบบที่ช่วยให้นักศึกษาสามารถ:ลงขาย, แลกเปลี่ยน, หรือแจกสิ่งของได้ง่าย
ค้นหาสิ่งของที่ต้องการ พูดคุยและสื่อสารระหว่างผู้ใช้งาน

Backend: PHP
Frontend: HTML, CSS, JavaScript
Database: MySQL

⭐ ฟีเจอร์หลัก
- สำหรับผู้ใช้ทั่วไป:ลงทะเบียน / เข้าสู่ระบบ
- เพิ่ม, แก้ไข, และลบรายการสิ่งของ
- ค้นหาสิ่งของพร้อมตัวกรอง
- ส่งข้อความคุยกับผู้อื่น
- ให้คะแนนและรีวิวหลังการแลกเปลี่ยน
- ดู Leaderboard ของผู้ใช้ยอดเยี่ยม

สำหรับแอดมิน:
- อนุมัติและจัดการรายการสิ่งของ
- จัดการผู้ใช้และหมวดหมู่
- ดูสถิติและรายงานปัญหา
